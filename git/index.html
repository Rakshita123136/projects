Git Tutorial

Getting Started with Git

Git is a distributed version control system that helps track changes in source code during software development.

Prerequisites:

  • Basic command line knowledge
  • Text editor installed
  • Git installed on your system

Check Git Version:

$ git --version

Git Basics

Learn the fundamental commands to start working with Git.

Essential Commands:

  • git init - Initialize a repository
  • git add - Stage changes
  • git commit - Save changes
  • git push - Upload to remote

Basic Workflow:

$ git init
$ git add .
$ git commit -m "Initial commit"
$ git push origin main

Git Branching

Branches allow you to develop features isolated from each other.

Branch Commands:

  • git branch - List branches
  • git checkout -b - Create new branch
  • git merge - Merge branches
  • git branch -d - Delete branch

Branching Example:

$ git checkout -b feature
$ git add .
$ git commit -m "Add new feature"
$ git checkout main
$ git merge feature

Git Flags and Options

Git commands can be modified with various flags to change their behavior. Here are the most commonly used flags for different Git commands.

git add flags

  • -A, --all - Stage all changes
  • -p, --patch - Interactively choose hunks to stage
  • -n, --dry-run - Show what would be added
  • -f, --force - Allow adding ignored files
  • -u, --update - Update tracked files

git commit flags

  • -m "message" - Add commit message
  • -a - Stage modified and deleted files automatically
  • --amend - Modify the last commit
  • -v - Show diff in commit message editor
  • --no-verify - Skip pre-commit hooks

git push flags

  • -u, --set-upstream - Set upstream for git pull/status
  • -f, --force - Force push changes
  • --tags - Push all tags
  • --dry-run - Show what would be pushed
  • --all - Push all branches

git pull flags

  • --rebase - Rebase instead of merge
  • --no-commit - Don't commit after merge
  • --squash - Squash all commits into one
  • -v, --verbose - Be more verbose
  • --allow-unrelated-histories - Merge unrelated branches

git branch flags

  • -d, --delete - Delete a branch
  • -D - Force delete branch
  • -m, --move - Rename a branch
  • -a, --all - List all branches
  • -r, --remotes - List remote branches

Common Usage Examples:

# Stage all changes and commit with message
$ git add -A
$ git commit -m "Update features"

# Force push to remote
$ git push -f origin main

# Create and switch to new branch
$ git checkout -b feature-branch

# Delete branch locally and remotely
$ git branch -d feature-branch
$ git push origin --delete feature-branch

Explore advanced Git features and techniques.

Advanced Features:

Advanced Commands:

$ git rebase main
